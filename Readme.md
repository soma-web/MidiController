# FAVOMI
## Fancy Voicemeter Mixer

Midi Controller to controll voicemeter

Parts:
- Linear Potentiometer 10k Ohm Linear
- Radial Potentiometer 10k Ohm
- Pushbutton
- Arduino Mega2560 R3
- Resistor 1k Ohmgit
