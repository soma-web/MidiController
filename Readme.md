# FAVOMI
## Fancy Voicemeter Mixer

Midicontroller to controll VoiceMeter https://vb-audio.com/

Parts:
- Linear Potentiometer 10k Ohm Linear
- Radial Potentiometer 10k Ohm
- Pushbutton
- Arduino Mega2560 R3
- Resistor 1k Ohmgit
